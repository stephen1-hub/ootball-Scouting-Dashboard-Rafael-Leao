# Football-Scouting-Dashboard-Rafael-Leao
# Overview

This project is a football analytics scouting dashboard built with Python and Streamlit. It analyzes the attacking profile of Rafael Leão using xG/xA metrics, shot profiling, and role-based performance analysis.

The dashboard is designed to simulate a professional club scouting interface (Opta/Wyscout-style), with a focus on decision-making, transition threat, shot selection, and tactical role interpretation.

It evaluates Leão’s performance in the 2025/26 Serie A season for AC Milan.

# Key Features

📈 Player performance overview (Goals, xG, xA, shots, xG/90, xA/90)
🧠 Role-based analysis (Starter vs Substitute impact on output)
🎯 Shot zone distribution (Penalty area, Six-yard box, Outside box)
🦶 Shot type breakdown (Right foot, Left foot, Head)
⚽ Situation analysis (Open play, Set pieces, Penalties)
📊 Efficiency metrics (xG per shot, zone efficiency, shot quality)
🧩 Decision-making insights (shot selection patterns by context)
🧠 Scouting insight panel (auto-generated tactical interpretation)
🎮 Dark-themed player dashboard UI (Opta-style layout simulation)

# Key Insights

High dependency on right-footed shooting actions in attacking phases
Majority of xG generated from penalty area zones (inside-box attacks)
Strong output in open play transition situations
Limited aerial contribution (low heading threat)
Shot profile heavily shaped by left-wing isolation scenarios
Moderate underperformance in some finishing phases vs xG
Reduced creative output compared to previous seasons (xA decline)

# Scouting Summary

Rafael Leão is a high-impact transition-based wide attacker whose output is driven by explosive ball progression and right-footed finishing actions.

He thrives in open-play situations where he can isolate defenders and attack the penalty area at speed. However, his shot profile remains heavily concentrated on a dominant foot and inside-channel movements, making him partially predictable against structured defensive blocks.

His overall value lies in progression, chaos creation, and penalty-area entry, rather than traditional striker efficiency or aerial dominance.
